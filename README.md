AirVista is a low-cost, IoT-enabled pulmonary function monitoring system designed to provide real-time respiratory and cardiopulmonary health assessment. Built using an ESP32 microcontroller, the system integrates the ADP810 differential pressure sensor for spirometry, the MAX30102 sensor for heart rate and SpO₂ monitoring, the SHT30 sensor for temperature and humidity measurement, and the MPU6050 IMU for motion detection.

The acquired respiratory data are processed to calculate key spirometric parameters, including Forced Vital Capacity (FVC), Forced Expiratory Volume in One Second (FEV1), and Peak Expiratory Flow (PEF), with BTPS correction applied for physiological accuracy. An offline-trained machine learning model analyzes the extracted features and classifies respiratory conditions into seven categories: Normal, Obstructive, Restrictive, Mixed, Cardiopulmonary Impairment, Tachycardia, and Bradycardia.

A Streamlit-based web dashboard displays real-time sensor data, respiratory waveforms, calculated parameters, prediction results, and confidence scores. AirVista demonstrates the integration of IoT, biomedical signal processing, and machine learning to create an affordable, portable, and intelligent respiratory health monitoring system for telemedicine, research, and educational applications.
